## 筛选
+ 通过 `column` 对象的 `filter`属性设置筛选功能<br>
+  `filterList` 设置你的筛选条件。包含`label`、`value`、`selected` 3 个属性<br>
+  `isMultiple` 开启筛选项多选，默认为 false<br>
+  `filterConfirm` 筛选确认函数<br>
+  `filterReset` 筛选重置函数<br>
